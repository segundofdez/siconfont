Siconfont

Siconfont is an icon font designed for all-purpose to use on the web under SIL Open Font license, currently composed for elements.

Nowadays there are plenty solutions to generate and use icon fonts. You can view some examples here.

Download Siconfont (v0.1.9) or view the project on Github.
